HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • These are considered list items.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • * is a wild card symbol, in which the rule(s) between brackets, are applied to ALL elements on the page.
  • A class selector will share a CSS rule with certain elements and only those elements.
  • Use a '.' before the class name to designate it as a class selector.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add file: adds a file to the next commit.
  • git commit -m "Message": commits the staged content/file as a new commit snapshot/version.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A method is used to execute code.
  • An argument is a value that the method uses.
  • Conditionals are used to compare data, such as if/else.
  • Uses the strict equality operator: "===".
  • True/False is determined from conditional statements.
  • For Loops can perform task(s) on each piece of data given, such as an array.
  • An array can store mulitple items with different data types, under a singular variable
  • Arrays use indexes for the location of a value, staring from [0].
